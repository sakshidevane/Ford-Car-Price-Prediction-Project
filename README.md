# Ford Car Price Prediction Project 🚗📊

---

## 📌 Overview

This repository contains my **Ford Car Price Prediction Project**, originally developed on Kaggle.

The goal of this project is to predict the price of Ford cars based on various features such as model, year, transmission type, mileage, fuel type, engine size, and more.

This project demonstrates a complete **Machine Learning workflow**, including data preprocessing, exploratory data analysis (EDA), feature engineering, model building, evaluation, and prediction.
